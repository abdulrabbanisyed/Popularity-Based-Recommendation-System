# Recommendation System
This is a movie recommendation system that uses pandas and its functions to analyze movie ratings and views. The system returns the top 10 movies based on rating and number of views.

## Features
- Returns top 10 movies based on rating.
- Returns top 10 movies based on the number of views.
- Uses pandas for data modeling and analysis.
